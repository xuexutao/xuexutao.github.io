基础命令

ls

ls命令显示指定目录的内容，默认参数为当前目录。

ls -l显示详细的列表

ls -F显示文件类型信息

cp

cp命令用来复制文件

cp file1 file2: 将文件file1复制到文件file2

cp file1 ... fileN dir : 将多个文件（file1 ... fileN）复制到目录dir

mv

mv命令有点类似cp，用来重命名文件

mv file1 file2 : 将文件名从file1重命名为file2

mv file1 ... fileN dir : 将多个文件移动到某个目录

torch

touch命令用来创建文件。如果文件已经存在，则该命令会更新文件的时间戳

toch file

rm

rm file : rm命令用来删除文件，文件一旦被删除通常无法恢复

rmdir

rmdir命令用来删除目录：

rmdir dir : 删除dir【但是要注意，这个只能删除空文件夹】

可以使用rm -rf来删除一个目录以及其中的所有内容

  • -r 会依次删除dir中的所有文件和子目录
  • -f 代表强制删除

echo

将它的参数显示到标准输出

grep

grep命令显示文件和输入流中和参数匹配的行

grep root /etc/passwd : 显示文件/etc/passwd中包含文本root的所有行

grep root /etc/* :

shell通配符

shell可以使用通配符来匹配文件名和目录名，比如*代表任意字符和数字

echo * :列出当前目录中的所有文件

  • at*展开为所有以at开头的文件名
  • *at展开为所有以at结尾的文件名
  • *at*展开为所有包含at的文件名

另外一个shell通配符问号（?）帮助shell确切匹配任意一个字符,如b?at与boat和brat相匹配

需要注意的是，shell是先展开通配符，然后执行命令行
