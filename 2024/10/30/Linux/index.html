在 Linux 虚拟机中安装并启动 SSH 服务（假设使用 OpenSSH）：

要使 Windows 主机能够访问 Linux 虚拟机的 22 号端口，可以在 VMware 中将网络模式设置为 桥接模式 或 NAT 模式：

  • 桥接模式：Linux 虚拟机会直接连接到与主机相同的网络，分配一个与主机在同一网段的 IP 地址，方便 Windows 直接访问。
  • NAT 模式：Linux 虚拟机会通过主机的 IP 地址访问外部网络。在这种模式下，需要进行端口转发，将主机的某个端口（如 2222）映射到虚拟机的 22 端口。

将本地文件传入到VM的虚拟机中，使用csp命令（速度超快！）: scp <本地文件路径> username@:<目标路径>
