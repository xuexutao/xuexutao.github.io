mac全局配置

之前一直用mac的终端去各种pip install 或者 git clone的时候总是特别慢，直到我在整blog时候真的忍受不了这样的龟速了。于是就有了vpn的全局配置，我只能说我好傻😭，怎么才知道！

1、我的是mac，所以直接添加配置即可

我的端口是7897。https://blog.csdn.net/w8y56f/article/details/120630022

我配置的这个，需要写在～/.zshrc，然后source ~/.zshrc即可配置全局代理 - export http_proxy=socks5://127.0.0.1:7897 - export https_proxy=socks5://127.0.0.1:7897

这样从mac 终端发出的请求就可以利用我的vpn了，下载github这些东西再也不会卡了～

安装mysql

  • 安装 MySQL brew install mysql
  • 启动 MySQL 服务器 brew services start mysql
  • 配置 MySQL 服务器 终端使用：mysql_secure_installation

在使用之前可以先进入mysql，看下 SHOW VARIABLES LIKE 'validate_password%';

看一下全局变量，其中有一个validate_password_length默认为8；将它改为四就可以设置常用的密码啦，不然会说不够强度

SET GLOBAL validate_password_length = 4;
