支持latex：

1、卸载原有的hexo-renderer-marked、安装pandoc插件

npm uninstall hexo-renderer-marked --save npm install hexo-renderer-pandoc --save

很重要的一个就是原有的pandoc模块会自动换行！！

解决：在node_modules-renderer-pandoc.js修改为var args = ['-M', 'pagetitle=dummy', '--wrap=none'];也就是添加上'--wrap=none'这个

2、安装hexo-filter-mathjax就可以了，然后在hexo的_config.yml加入
