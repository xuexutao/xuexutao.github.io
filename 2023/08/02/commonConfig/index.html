Linux安装常用软件

1、安装java

安装步骤：

  • 执行命令yum -y list java*查看可安装java版本
  • yum install -y java-1.8.0-openjdk-devel.x86_64， 一键安装
  • 安装位置：/usr/lib/jvm

~/bigdata/hadoop-2.7.5/tmp

2、安装redis

安装步骤：

  • wget http://download.redis.io/releases/redis-6.0.8.tar.gz
  • tar xzf redis-6.0.8.tar.gz
  • cd redis-6.0.8
  • make

执行完 make 命令后，redis-6.0.8 的 src 目录下会出现编译后的 redis 服务程序 redis-server，还有用于测试的客户端程序 redis-cli。

启动 redis 服务 客户端

  • cd src
  • ./redis-server 启动客户端: ./redis-cli

3、安装docker

1、使用 yum 安装

安装步骤

  1. 配置yum软件源
  2. 安装依赖包
  3. 安装
  1. 执行下面的命令添加 yum 软件源：

  • sudo yum-config-manager –add-repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

  • sudo sed -i ‘s/download.docker.com/mirrors.aliyun.com/docker-ce/g’ /etc/yum.repos.d/docker-ce.repo

官方源

  • sudo yum-config-manager –add-repo https://download.docker.com/linux/centos/docker-ce.repo
  1. 执行以下命令安装依赖包：sudo yum install -y yum-utils

如果需要测试版本的 Docker 请执行以下命令：
sudo yum-config-manager –enable docker-ce-test

  1. 安装 Docker 更新 yum 软件源缓存，并安装 docker-ce。
    • sudo yum install docker-ce docker-ce-cli containerd.io

2、 启动 Docker
sudo systemctl enable docker

sudo systemctl start docker

如果拉取缓慢配置国内源：https://yeasy.gitbook.io/docker_practice/install/mirror

CentOS8 额外设置
由于CentOS8 防火墙使用了nftables，但Docker尚未支持nftables，可以使用如下设置使用iptables：

更改 /etc/firewalld/firewalld.conf中的FirewallBackend=nftables改为FirewallBackend=iptables
